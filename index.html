Manual de Usuario del Sistema Interactivo para el Aprendizaje de Vectores

Introducción

El sistema interactivo desarrollado permite a los estudiantes aprender conceptos y operaciones vectoriales mediante el uso de un robot graficador y una interfaz interactiva. Este manual está diseñado para guiar al usuario en la configuración, operación y mantenimiento del sistema.

Componentes del Sistema

  • Robot Graficador (KINEMATIC):
    • Microcontrolador ESP32.
    • Motores N20 con ruedas.
    • Baterías recargables Li-ion.
    • Estructura de soporte impresa en 3D.
    • Sensores integrados para movimiento y ángulos.
  • Interfaz Interactiva:
    • Software basado en navegador.
    • Gráficos interactivos para representar vectores.
    • Opciones para ingresar valores de vectores y realizar operaciones.
  • Accesorios Complementarios:
    • Cable USB para cargar las baterías.
    • Manual impreso de referencia rápida.

Instrucciones de Configuración

  1. Preparar el Robot Graficador:
    • Asegúrese de que las baterías estén completamente cargadas.
    • Verifique que las ruedas estén correctamente ajustadas.
    • Coloque el robot en una superficie lisa y marcada con puntos de referencia (como baldosas).
  2. Configurar la Interfaz Interactiva:
    • Acceda al software mediante un navegador web compatible.
    • Conecte el ESP32 del robot al software utilizando la red Wi-Fi configurada.
    • Ingrese al menú principal y seleccione la opción "Configurar Robot".
  3. Sincronizar el Sistema:
    • Una vez conectado, la interfaz mostrará un mensaje de "Robot Conectado".
    • Realice una prueba de movimiento desde la opción "Prueba de Movimiento" para confirmar la conexión.

Uso del Sistema

  1. Introducir Vectores:
    • Desde la interfaz, seleccione la opción "Ingresar Vectores".
    • Ingrese las coordenadas X e Y del vector deseado.
    • Realice operaciones como:
      • Suma de vectores.
      • Multiplicación por escalares.
  2. Graficar Vectores:
    • Haga clic en "Graficar" para visualizar el vector ingresado en el plano cartesiano.
    • El robot se moverá en la trayectoria correspondiente al vector.
  3. Operaciones Competitivas:
    • Forme equipos y compita para completar operaciones vectoriales en el menor tiempo posible.
    • Utilice el robot para ejecutar operaciones como suma y resta de vectores.
    • Registre los tiempos y resultados en la interfaz.

Mantenimiento del Sistema

  • Robot Graficador:
    • Limpieza: Asegúrese de limpiar las ruedas y la estructura después de cada sesión para evitar acumulación de suciedad.
    • Cuidado de los Motores: Evite exceder el voltaje nominal de 6V para prevenir daños en los motores.
  • Interfaz Interactiva:
    • Mantenga actualizado el software para garantizar un rendimiento óptimo.
    • Verifique periódicamente la conexión Wi-Fi del ESP32.
  • Reemplazo de Componentes:
    • Reemplace las baterías cuando su capacidad disminuya notablemente.
    • En caso de daño en las piezas impresas en 3D, use los planos proporcionados para reimprimirlas.

Resolución de Problemas

  • Problema: El robot no se conecta al software
    Solución: Verifique que el ESP32 esté encendido y dentro del rango de la red Wi-Fi.
  • Problema: Movimiento irregular del robot
    Solución: Revise las ruedas y asegúrese de que no haya objetos obstruyendo el movimiento.
  • Problema: El gráfico no corresponde al vector
    Solución: Confirme que las coordenadas ingresadas sean correctas en la interfaz.
